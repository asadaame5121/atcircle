# アプリケーション概要：AT-Circle (旧 Bluesky Webring)

## 1. プロジェクトの概要

**AT-Circle**は、次世代分散型SNSプロトコルである**AT-Proto
(Bluesky)**のエコシステムに特化した、モダンな**ウェブリング（Webring）兼コンテンツアグリゲーター**プラットフォームです。

かつての個人サイト文化にあった「ウェブリング」の仕組みを、最新の認証技術（OAuth）とコンテンツ配信（RSS/Atom）で再構築しました。Blueskyユーザーが独立した個人のウェブサイトやブログを持ち寄り、相互にトラフィックを送り合い、コミュニティを形成することを支援します。

## 2. コア機能

- **Bluesky連携 (OAuth認証):**
  Blueskyのアカウントを使って、安全かつ簡単にログイン・登録が可能です。
- **リング（サークル）作成・管理:**
  特定の趣味やテーマでリングを作成し、メンバーを招待・承認制で管理できます。
- **ナビゲーションウィジェット:**
  参加サイトのフッターなどに埋め込めるJSウィジェットを提供。「前へ」「次へ」「ランダム」でリング内のサイトを回遊できます。
- **アンテナ（RSS集約）:**
  参加メンバーのサイトからRSSフィードを自動検出し、最新記事を一つのタイムラインとして閲覧できる「アンテナ機能」を搭載。
- **OPMLエクスポート:**
  リング内の全サイトの購読リストを一括ダウンロードでき、お気に入りのRSSリーダーへの移行を容易にします。
- **ウィジェットビルダー:**
  デザインをカスタマイズ可能なビジュアルエディタを搭載。

## 3. ターゲット層

- **個人ブロガー・技術者:**
  Blueskyを利用しており、自身のドメインで発信を行っている層。
- **クリエイター:**
  イラストレーター、写真家、同人作家など、作品をポートフォリオサイトで公開している層。
- **ニッチコミュニティ:**
  分散型プロトコルや、特定の趣味（技術、読書、創作など）に関心の高いテックネイティブな層。

## 4. 技術スタック

- **Backend/Frontend:** Hono, Vite, TypeScript
- **Database:** SQLite (`node:sqlite`)
- **Infrastructure:** Fly.io (Dockerベース)
- **Protocol:** AT-Proto (Bluesky API)
